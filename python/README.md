# WRIVON — Python Implementation

Python port of the WRIVON CLI AI coding agent.

## Quick Start

```bash
pip install -e .
wrivon setup
wrivon start
```

## Development

```bash
pip install -e ".[dev]"
pytest
```

## Commands

- `/help`, `/model`, `/diff`, `/commit`, `/push`, `/review`, `/refactor`
- `/status`, `/map`, `/sessions`, `/resume`, `/export`, `/build`
- `/clear`, `/exit`
